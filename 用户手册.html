1. 软件介绍

这是一款基于游戏《Fate/Grand Order》中Lancer因陀罗制作的同人桌宠。桌宠无盈利，一切版权归于原版权方。

除了提交给llm api的数据之外，一切用户数据（聊天记录、屏幕截图等）存储于本地。想要使用对话功能，你需要自备deepseek、chatgpt或者硅基流动等服务商的api url和key。

2. 安装步骤

步骤1：确保电脑安装 Python 3.1 及以上版本（官网：https://www.python.org/）；

步骤2：下载桌宠项目压缩包，解压到任意文件夹；

步骤3：打开终端/命令行，进入项目根目录，执行以下命令安装依赖：

pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

步骤4：运行 src/main.py 启动软件。

3. 基本使用

3.1 拖动桌宠

按住鼠标左键拖动桌宠，可自由调整其在屏幕上的位置。

3.2 与桌宠聊天

双击桌宠，会弹出聊天框，输入文字后发送，桌宠会回复消息。

3.3 屏幕监视功能

方式1：右键点击托盘图标，选择「屏幕监视：开启/关闭」，实时切换功能；

方式2：打开设置界面，勾选「启用屏幕监视」，设置间隔时间后保存；

功能说明：开启后，桌宠会按设定间隔截图，自动评论屏幕内容（需配置API密钥）。

4. 设置说明

右键点击托盘图标，选择「设置」，可调整以下选项：

  • 桌宠缩放：调整立绘大小（0.1-5.0倍）；
  • 待机间隔：设置多久无操作后播放待机动画；
  • 屏幕监视：开启/关闭定时截图评论，设置间隔时间；
  • 气泡时长：设置桌宠头顶临时消息的显示时间。

6. 常见问题

Q1：启动后看不到桌宠？
A1：检查 assets/images/idle 文件夹是否有PNG动画帧；查看终端是否有报错（如路径错误、依赖缺失）。
Q2：屏幕监视功能不生效？
A2：确保已配置API密钥（在设置界面填写）；检查网络连接；确认功能已通过托盘或设置开启。
Q3：桌宠大小异常？
A3：在设置中调整「桌宠缩放」参数，建议设置为0.8-1.2倍。
